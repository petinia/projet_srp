## REPONSES
## machine de Turing 
Proposée en 1936 par Alan Turing, la machine de Turing est un modèle théorique très simple, conçu pour répondre à une question fondamentale.

## L'IA générative
Une IA générative est un modèle capable de produire du contenu nouveau: texte, image, musique, voix, vidéo, code informatique, etc.

## L'IA non générative
Une IA non générative est un système qui ne crée pas de nouveau contenu, mais qui analyse, classe, détecte, reconnaît ou prédit des informations à partir de données existantes.
